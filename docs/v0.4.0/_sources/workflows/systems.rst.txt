Flow-cytometer systems
======================

This page documents the detector-system presets used by RosettaX in the
scattering workflow. Each card represents one supported system family and shows
available 3D detector representations.

When a figure is not available yet, a placeholder is shown so the missing views
can be filled in without changing page structure.
