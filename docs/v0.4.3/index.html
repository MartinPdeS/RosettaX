RosettaX documentation#

RosettaX calibrates exported FCS data for fluorescence and scattering workflows and then reapplies saved calibrations to experimental files. This reference focuses on practical workflow guidance and the modeling details behind each output.

Date: Jul 09, 2026
Version: 0.4.3
